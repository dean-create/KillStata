A collection of community projects built on Killstata.

:::note
Want to add your Killstata related project to this list? Submit a PR.
:::

You can also check out [awesome-killstata](https://github.com/awesome-killstata/awesome-killstata) and [killstata.cafe](https://killstata.cafe), a community that aggregates the ecosystem and community.
