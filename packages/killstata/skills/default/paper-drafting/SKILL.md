# Paper Drafting

Use this after `research_brief` and baseline outputs exist.

## Tool path

1. Resolve the saved `research_brief.json`.
2. Resolve the baseline result directory or published result artifact.
3. Optionally resolve the heterogeneity bundle.
4. Call `paper_draft`.

## Rules

- Do not invent unsupported numbers or references.
- If some evidence is missing, leave a clear `待补充` placeholder.
- Keep the draft aligned with saved structured artifacts from the same run.
