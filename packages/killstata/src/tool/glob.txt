- Fast file pattern matching tool for finding data files in the working directory
- Supports glob patterns like "**/*.xlsx" or "data/**/*.csv"
- Returns matching file paths sorted by modification time
- Use this tool to locate data files (csv/xlsx/dta/parquet) or artifacts by name pattern
- When you are doing an open-ended search that may require multiple rounds of globbing and grepping, use the Task tool instead
- You have the capability to call multiple tools in a single response. It is always better to speculatively perform multiple searches as a batch that are potentially useful.
